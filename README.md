## Descripción
Implementación del algoritmo de cifrado Hill que permite encriptar y desencriptar mensajes usando una matriz clave cuadrada. Incluye una interfaz web interactiva.

# Como se usa
Escribe un mensaje o una oracion
Ingresa la matriz clave 2x2
Haz click en encriptar o en desencriptar

# Bases Matematicas
Analfabeto usado: A-Z
Operacion: C = (P x K)
Desencriptar: P = (C x K-1)
